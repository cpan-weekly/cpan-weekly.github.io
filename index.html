CPAN Weekly

CPAN Weekly is a mailing list for Perl 5 programmers.
Each week you'll get an email with a short description of, and example usage for, a different CPAN module.

Why?

So next time you hit a certain problem, you might think "I read about a module for that", instead of "I could write a module for that" :-)

Sign up

Or you can just send a blank email to newsletter-subscribe@cpan-weekly.org.

You can help!
